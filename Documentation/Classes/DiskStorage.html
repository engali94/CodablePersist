DiskStorage

public class DiskStorage<T> where T : Identifiable, T : Decodable, T : Encodable
extension DiskStorage: CacheStorable

Disk Storage utilizes FileManager to cache and retrive data to and from the disk it gives a convenient and easy to use way to cache any Codable and Identifiable object.
